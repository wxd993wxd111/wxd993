# ClipAI - AI智能视频剪辑与多平台发布系统

🎬 一款现代化的AI智能视频剪辑工具，支持一键生成适配抖音、视频号、小红书、快手等平台的文案和背景音乐，并实现多平台一键发布。

## ✨ 核心功能

### 📤 视频上传
- 支持拖拽上传
- 支持多种视频格式（MP4, MOV, AVI, WebM）
- 视频预览和信息提取
- 最大支持 500MB 视频文件

### 🤖 AI智能剪辑
- **AI文案生成**：根据不同平台风格自动生成爆款标题、描述文案和话题标签
  - 🎵 **抖音**：年轻化、爆款标题、悬念式、情感共鸣
  - 📺 **视频号**：情感故事、生活分享、知识干货
  - 📕 **小红书**：种草风格、emoji丰富、实用性强
  - ✊ **快手**：接地气、真实感、社区互动

- **AI音乐匹配**：根据视频内容和情绪自动匹配背景音乐
  - 多种情绪分类：欢快、感人、励志、悬疑、浪漫、酷炫
  - 平台热门音乐推荐

### 🚀 一键发布
- 多平台账号管理
- 一键发布到抖音、视频号、小红书、快手
- 发布状态追踪
- 草稿箱管理

## 🛠️ 技术栈

- **前端**：React 18 + TypeScript
- **UI框架**：Ant Design 5
- **状态管理**：Zustand
- **样式**：Tailwind CSS
- **构建工具**：Vite

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

## 🔧 平台API接入

各平台开放平台需要申请资质和审核：

| 平台 | 开放平台 | 申请条件 |
|------|---------|---------|
| 抖音 | [open.douyin.com](https://open.douyin.com) | 企业资质或个人创作者认证 |
| 视频号 | 微信公众平台 | 完成创作者认证 |
| 小红书 | 小红书创作中心 | 入驻专业号 |
| 快手 | [open.kuaishou.com](https://open.kuaishou.com) | 开发者账号注册 |

> 当前版本提供本地草稿生成和导出功能，正式发布功能需要接入各平台API。
